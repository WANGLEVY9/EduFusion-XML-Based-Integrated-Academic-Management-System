# 基于 XML 的异构数据库集成教务系统 - 实施说明

## 1. 项目目标
- 跨学院课程共享
- 跨学院选课与退课
- 集成端统一统计
- XML + XSD + XSLT 数据交换流程展示

## 2. 当前版本实现范围
- 多模块 Maven 工程搭建
- 三学院服务端基础网关（A/B/C）
- 集成服务器核心逻辑
- 客户端 Swing 登录入口
- XML 请求示例、XSD 校验规则、XSLT 转换模板
- SQL 建表示例脚本

## 3. 后续扩展建议
- 将 InMemoryCollegeStore 替换为真实 JDBC DAO 层
- 引入 Socket/HTTP 通信替代文件轮询
- 完成完整学生端/管理员端业务界面
- 增加统计图表展示（JFreeChart）
- 增加日志与异常跟踪
